# 🧩 Sudoku em Java

Projeto de um jogo de Sudoku desenvolvido em Java com interface gráfica Swing, como parte de um desafio da DIO (Digital Innovation One).

## 🚀 Funcionalidades

- Interface gráfica interativa
- Validação de jogadas em tempo real
- Verificação de status do jogo
- Detecção de erros no tabuleiro
- Reset do jogo com confirmação
- Células fixas e editáveis

## 🛠️ Tecnologias

- Java
- Swing

## 🧠 Conceitos aplicados

- Programação Orientada a Objetos (POO)
- Separação de responsabilidades
- Manipulação de eventos (Observer)
- Estrutura em camadas (model, service, ui)
